Оплата

Работаем по договору с НДС 20%

  • Для юридических лиц
    Оплата для юридических лиц осуществляется безналичным способом по предварительно выставленному счету.
  • ЭДО
    Работаем по ЭДО в системе СБИС, возможно подключение других операторов.
